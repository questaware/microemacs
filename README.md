# microemacs
## My microemacs <h2> tag

**In Windows:**

For Visual Studio 2006:

cd wincon/release

gmake

For Visual Studio 2019:

Open Project/Solution vs2019/memacs.sln

**In Linux**

cd unix

make

**Installation**

Put the executable, (macros/emacs.rc or macros/.emacsrc), src/emacs.md
in a directory on the path.
